# 物见 v1.0.1

这是一个修复版本，重点解决 PDF 导出中物品图片没有稳定显示的问题。

## 下载哪个 APK

| 文件 | 适用设备 | 建议 |
| --- | --- | --- |
| `icheck-v1.0.1-arm64-v8a.apk` | 绝大多数现代 Android 手机 | 推荐下载 |
| `icheck-v1.0.1-armeabi-v7a.apk` | 较老的 32 位 Android 手机 | 老设备备用 |
| `icheck-v1.0.1-x86_64.apk` | Android 模拟器或 x86_64 设备 | 模拟器测试 |

不确定手机架构时，可以用：

```powershell
adb shell getprop ro.product.cpu.abilist
```

包含 `arm64-v8a` 就下载 `icheck-v1.0.1-arm64-v8a.apk`。

## 本版本内容

- 修复 PDF 导出时图片卡片可能没有图片的问题。
- PDF 导出现在会先读取物品图片并显式嵌入为 `MemoryImage`，避免 layout 阶段再读文件导致图片缺失。
- 固定 PDF 图片预览区域尺寸，提升导出布局稳定性。
- 保留 v1.0.0 的所有功能：拍照识别、待确认队列、物品视图、PDF/Excel/Markdown 导出、多配置、token 统计和存储管理。

## 安装

```powershell
adb install -r icheck-v1.0.1-arm64-v8a.apk
```

如果提示签名不一致，需要先卸载旧包再安装；卸载会删除本地数据：

```powershell
adb uninstall com.wujian.app.icheck
adb install icheck-v1.0.1-arm64-v8a.apk
```

## 首次使用

1. 安装后打开应用，允许相机权限。
2. 进入“设置”，填写 Base URL、API Key 和模型 ID。
3. 点击“测试连接”。
4. 回到“主页”拍照识别。

默认配置：

- Base URL：`https://ark.cn-beijing.volces.com/api/v3`
- 模型 ID：`doubao-seed-2-0-mini-260428`

## 校验

下载后可以用 `SHA256SUMS.txt` 校验 APK 是否完整：

```powershell
Get-FileHash .\icheck-v1.0.1-arm64-v8a.apk -Algorithm SHA256
```

## 注意

当前 APK 使用 debug 签名构建，适合个人安装和测试下载；正式上架应用商店前需要配置 release keystore。
