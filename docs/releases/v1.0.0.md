# 物见 v1.0.0

这是物见的第一个 Android release。它面向个人家庭物品整理场景：拍照记录物品，用火山方舟视觉模型识别信息，确认后保存为本地清单，并支持导出。

## 下载哪个 APK

| 文件 | 适用设备 | 建议 |
| --- | --- | --- |
| `icheck-v1.0.0-arm64-v8a.apk` | 绝大多数现代 Android 手机 | 推荐下载 |
| `icheck-v1.0.0-armeabi-v7a.apk` | 较老的 32 位 Android 手机 | 老设备备用 |
| `icheck-v1.0.0-x86_64.apk` | Android 模拟器或 x86_64 设备 | 模拟器测试 |

不确定手机架构时，可以用：

```powershell
adb shell getprop ro.product.cpu.abilist
```

包含 `arm64-v8a` 就下载 `icheck-v1.0.0-arm64-v8a.apk`。

## 本版本内容

- 支持单张拍摄识别和连续拍摄入队。
- 支持待确认队列，识别结果可编辑后入库。
- 支持按名称、分类、房间、箱号搜索和筛选。
- 支持 PDF、Excel、Markdown 导出。
- 支持多个 API/模型配置。
- 支持连接测试、token 统计和本地存储占用统计。
- release 包已包含 `android.permission.INTERNET`，修复旧 release 无法联网导致的 `SocketFailed host lookup` 问题。

## 安装

```powershell
adb install -r icheck-v1.0.0-arm64-v8a.apk
```

如果提示签名不一致，需要先卸载旧包再安装；卸载会删除本地数据：

```powershell
adb uninstall com.wujian.app.icheck
adb install icheck-v1.0.0-arm64-v8a.apk
```

## 首次使用

1. 安装后打开应用，允许相机权限。
2. 进入“设置”，填写 Base URL、API Key 和模型 ID。
3. 点击“测试连接”。
4. 回到“主页”拍照识别。

默认配置：

- Base URL：`https://ark.cn-beijing.volces.com/api/v3`
- 模型 ID：`doubao-seed-2-0-mini-260428`

## 校验

下载后可以用 `SHA256SUMS.txt` 校验 APK 是否完整：

```powershell
Get-FileHash .\icheck-v1.0.0-arm64-v8a.apk -Algorithm SHA256
```

## 注意

当前 APK 使用 debug 签名构建，适合个人安装和测试下载；正式上架应用商店前需要配置 release keystore。
