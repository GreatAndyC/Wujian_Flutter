# 物见 v1.0.2

这个版本把拍摄、确认、视图管理和导出流程一起打磨了一轮，重点是减少拍照确认步骤、补齐批量操作，以及让导出更像一个真正可落地使用的工具。

## 下载哪个 APK

| 文件 | 适用设备 | 建议 |
| --- | --- | --- |
| `icheck-v1.0.2-arm64-v8a.apk` | 绝大多数现代 Android 手机 | 推荐下载 |
| `icheck-v1.0.2-armeabi-v7a.apk` | 较老的 32 位 Android 手机 | 老设备备用 |
| `icheck-v1.0.2-x86_64.apk` | Android 模拟器或 x86_64 设备 | 模拟器测试 |

不确定手机架构时，可以用：

```powershell
adb shell getprop ro.product.cpu.abilist
```

包含 `arm64-v8a` 就下载 `icheck-v1.0.2-arm64-v8a.apk`。

## 本版本内容

- 新增应用内相机，拍照后直接加入后台识别队列，不再弹出系统相机的对勾/叉确认页。
- 待确认队列列表增加缩略图，进入确认编辑时也会显示图片预览。
- 视图页支持长按选择、批量删除。
- 物品详情页调整了底部安全区，避免被 Android 底部导航遮挡。
- 导出支持两种去向：分享给其他应用，或通过系统文件选择器直接保存到本地。
- PDF 封面去掉了顶部那几个无意义的空白圆角块，封面信息更简洁。

## 安装

```powershell
adb install -r icheck-v1.0.2-arm64-v8a.apk
```

如果提示签名不一致，需要先卸载旧包再安装；卸载会删除本地数据：

```powershell
adb uninstall com.wujian.app.icheck
adb install icheck-v1.0.2-arm64-v8a.apk
```

## 校验

下载后可以用 `SHA256SUMS.txt` 校验 APK 是否完整：

```powershell
Get-FileHash .\icheck-v1.0.2-arm64-v8a.apk -Algorithm SHA256
```

## 注意

当前 APK 使用 debug 签名构建，适合个人安装和测试下载；正式上架应用商店前需要配置 release keystore。
